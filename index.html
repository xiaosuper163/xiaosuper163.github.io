Projects

Vancouver Housing Market Decoder

In this project, we implemented the web scraping script to collect selling prices and detailed information of housing properties. Entity resolution was applied to combine the data from different data sources. We applied various machine learning algorithms to predict the listing price and selling price. Based on the machine learning model, we built an interactive web application with python Flask framework.

Quora Insincere Question Classification

We implemented a deep LSTM model with Keras to classify whether a question is sincere or not. To incorporate the manually created features, we used the multi-input LSTM model. We ranked at top 15% out to 2000 teams at the moment we submitted our best submission.

Yelp Dataset Analysis

We applied gensim Latent Dirichlet Allocation (LDA) to the Yelp public dataset to extract the topics from the text review. We also built a scalable recommendation system based on the numerical ratings of the restaurant business with Spark Mllib. To make the results available to the users, we built an interactive web application. We also showed some interesting findings with visualizations.

Pain Level Recognition

based on the bio-sensor signals, we experimented with various machine learning and deep learning algorithms to classify pain level. The best model achieved better accuracy than published article.

Recommendation System

We designed a recommendation system to predict numerical ratings; ranked the 2nd place in kaggle in-class competition leader board.

Online Document-Clustering System for Big Data

We deployed a Cloudera manager on a five-node cluster. With the cluster, we built a scalable online document-clustering system by using Kafka and Spark Streaming.
